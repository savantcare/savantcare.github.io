# How to write a component

3 core concepts:

  1. The entrance file is: search-interfaces.vue This component gives the search terms for the search dropdown by getting imported then defined and then mounted inside vl-search-box/index.vue For the flow chart see vl-search-box

  2. Each component will usually have sub components for vl and cl. Terms like vl and cl are explained in glossary. Which subcomponent is invoked is based on search term the keys ctToShowInCSVL and ctToShowInCL inside search-interfaces.vue

  3. Data in a component is CRUD through a model. During CRUD the status of row is maintained inside rowstatus.js
